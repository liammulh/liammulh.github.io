antigravity.py is an Easter egg module in Python's source code.
Published: Thursday, October 5, 2023
Tagged under: python

I have Python's reference implementation, cpython, checked out on my computer. (I occasionally peruse the source code to see how things work under the hood.) Today, I noticed a module with a strange name: antigravity.py [1]. It's an Easter egg. It contains code that opens your browser to [2]. It also contains an implementation of Randall Munroe's geohashing algorithm described in [3]. I looked at the Git history for antigravity.py, and apparently it's been in cpython since October 15th, 2008. I recently stumbled across xkcd 426, so stumbling across antigravity.py and seeing an implementation of geohashing made me chuckle.
